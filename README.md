# پروژه آوای یقین (Avaye Yaghin)
پاسخگوی هوشمند به سوالات دینی با طراحی شیشه‌ای مسجدی (Islamic Glassmorphism)

## ساختار فایل‌ها
- `index.html`: رابط کاربر، صفحه ورود، محیط چت و مودال پنل مدیریت
- `styles.css`: استایل‌های Tailwind و افکت‌های glassmorphism
- `app.js`: منطق فرانت‌اند، ورود کاربران، رمز مدیریت مخفی (`Amidhjsos62627@_897`) و مدیریت حافظه
- `functions/api/chat.js`: بک‌اند Serverless کلودفلر جهت ارتباط امن با API
